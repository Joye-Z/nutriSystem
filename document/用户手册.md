# 健康食谱管理与膳食规划系统
## 用户手册

## 1. 文档概述

### 1.1 文档目的
本文档旨在为健康食谱管理与膳食规划系统的最终用户提供全面的使用指南，帮助用户快速上手系统功能，正确使用各项功能进行健康食谱管理和个性化膳食规划。

### 1.2 适用范围
本文档适用于以下用户群体：
- 系统注册用户
- 健康饮食爱好者
- 希望通过科学膳食改善健康状况的用户
- 需要进行膳食规划和营养管理的用户

### 1.3 术语定义
| 术语 | 解释说明 |
|------|----------|
| BMI | 体重指数（Body Mass Index），用于评估人体胖瘦程度和健康状况 |
| BMR | 基础代谢率（Basal Metabolic Rate），维持身体基本功能所需的最低能量消耗 |
| TDEE | 总能量消耗（Total Daily Energy Expenditure），包括基础代谢、活动消耗等在内的每日总能量消耗 |
| 膳食计划 | 用户根据个人需求制定的每日/每周餐饮安排 |
| 食谱 | 包含食材、烹饪步骤、营养信息等的完整烹饪指南 |
| 营养分析 | 对食物或膳食的营养成分进行量化分析的过程 |

## 2. 系统概述

### 2.1 系统简介
健康食谱管理与膳食规划系统是一款专注于健康饮食管理的综合性平台，旨在帮助用户发现健康食谱、制定个性化膳食计划，并通过科学的营养分析指导用户健康饮食。

### 2.2 主要功能特点
- **丰富的食谱资源**：提供多样化的健康食谱，涵盖不同菜系、不同健康目标和不同烹饪难度
- **个性化膳食规划**：根据用户健康目标和饮食偏好，制定个性化的每日/每周膳食计划
- **全面的营养分析**：提供详细的营养成分分析，帮助用户了解膳食的营养价值
- **科学的健康建议**：基于用户的健康数据和饮食记录，提供个性化的健康饮食建议
- **便捷的用户管理**：支持用户信息管理、健康数据记录和饮食偏好设置

### 2.3 系统运行环境
- **浏览器要求**：推荐使用 Chrome 90+、Firefox 88+、Safari 14+ 或 Edge 90+ 浏览器
- **分辨率要求**：建议使用 1366x768 及以上分辨率，最佳体验为 1920x1080
- **网络要求**：需要稳定的互联网连接以获取最新食谱和进行数据同步

## 4. 功能模块使用指南

### 4.0 首页仪表盘

登录系统后，用户将首先看到首页仪表盘，其中包含以下内容：

#### 4.0.1 欢迎信息
系统会显示欢迎信息，问候当前登录用户，并提醒用户完成膳食计划。

#### 4.0.2 一周卡路里趋势图
首页展示了"本周每日膳食计划卡路里"趋势图，该图表具有以下功能：

1. 以折线图形式展示本周（从周日到周六）的每日卡路里摄入情况
2. 每个数据点显示具体的卡路里数值

该趋势图数据来源于用户的膳食计划，系统会自动计算并展示每日的卡路里合计，帮助用户直观地了解一周的卡路里摄入趋势。

### 4.1 个人中心

#### 4.1.1 基本信息管理
1. 登录系统后，点击右侧「个人资料」
2. 在「个人」页面查看和修改个人资料：
   - 头像：点击上传新头像
   - 账号：查看账号名称
   - 名称：可以修改名称
   - 电话：可以填写电话信息
   - 邮箱：可以填写邮箱信息
3. 点击「保存」按钮更新信息

#### 4.1.2 健康档案管理
1. 在在健康食谱管理与膳食规划系统中，选择「营养分析」页面
2. 填写或更新以下健康数据：
   - 身高（厘米）
   - 体重（千克）
   - 年龄
   - 性别
   - 活动水平（久坐、轻度活动、中度活动、重度活动、非常活跃）
   - 健康目标（维持健康、减重、增肌、改善体能等）
3. 点击保存健康数据、计算营养需求，系统会自动计算BMI值和营养需求分析（基础代谢率、总能量消耗、推荐卡路里）

#### 4.1.3 账号安全设置
1. 在系统右侧，选择「修改密码」
2. 执行以下操作：
   - 修改密码：输入新密码，确认新密码，点击保存完成修改
   - 系统使用MD5加盐方式进行密码加密存储

### 4.2 食谱浏览与搜索

#### 4.2.1 食谱浏览
1. 登录系统后，右侧选择「食谱中心」
2. 浏览各式食谱
3. 上下滚动页面、分页选择查看更多食谱
4. 点击食谱卡片查看食谱详情

#### 4.2.2 食谱分类浏览
1. 在「食谱中心」顶部导航栏点击「分类」按钮
2. 在分类页面查看系统提供的所有食谱分类：
   - 家常菜
   - 健康餐
   - 减脂餐
   - 荤菜
   - 素菜
3. 点击相应分类查看该分类下的食谱列表

#### 4.2.3 食谱搜索
1. 在顶部导航栏的搜索框中输入关键词（食谱名称）、选择相关分类、难度
2. 按下回车键或点击搜索图标进行搜索
3. 在搜索结果页面查看符合条件的食谱

### 4.3 食谱详情

#### 4.3.1 查看食谱详情
1. 点击食谱卡片进入食谱详情页面
2. 查看食谱的以下内容：
   - 食谱标题、图片和简介
   - 所需时间、难度级别
   - 卡路里和营养成分信息
   - 详细的食材清单和用量
   - 步骤化的烹饪方法

### 4.4 膳食计划管理

#### 4.4.1 创建膳食计划
1. 点击右侧的「膳食规划」按钮
2. 在膳食计划页面点击「新增计划」按钮
3. 填写计划信息：
   - 计划名称（如「健康减重计划」）
   - 计划日期
   - 餐次类型
   - 选择食谱
   - 自定义餐食
   - 热量
   - 备注
4. 点击「确定」按钮完成新增膳食计划

#### 4.4.2 添加食谱到计划
1. 在膳食计划详情页面，选择具体日期
2. 若没有膳食计划，可选择添加第一个计划；若已有膳食计划，也可对其进行添加、修改、删除操作。
3. 填写计划信息：
   - 计划名称（如「健康减重计划」）
   - 计划日期
   - 餐次类型
   - 选择食谱
   - 自定义餐食
   - 热量
   - 备注
4. 重复以上步骤，为每个日期和餐食类型添加膳食计划

#### 4.4.3 查看营养分析
1. 在系统右侧，点击「营养分析」页面
2. 输入对应的个人健康数据
3. 查看以下营养分析数据：
   - 每日/每周卡路里摄入趋势图
   - 蛋白质、碳水化合物、脂肪的摄入比例
   - 维生素、矿物质等微量营养素摄入情况
   - 与推荐摄入量的对比分析
4. 根据分析结果调整膳食计划

### 4.5 营养分析

系统将根据您的健康档案数据自动计算营养需求，并提供个性化的膳食建议。

## 5. 常见问题与故障排除

### 5.1 账号问题

#### 5.1.1 无法登录系统
**问题描述**：输入正确的用户名和密码后，无法成功登录系统。

**解决方案**：
1. 检查用户名和密码是否输入正确，注意区分大小写
3. 清除浏览器缓存和Cookie后重新登录
4. 确认账号状态是否正常，如有异常请联系系统人员

### 5.2 功能使用问题

#### 5.2.1 无法上传头像
**问题描述**：在个人中心上传头像时失败。

**解决方案**：
1. 检查图片格式是否为JPG、PNG、GIF等支持的格式
2. 确认图片大小是否在系统限制范围内（通常为10MB以内）
3. 检查网络连接是否稳定
4. 尝试使用不同浏览器或刷新页面后再次尝试

#### 5.2.2 无法创建膳食计划
**问题描述**：点击「创建膳食计划」后，系统无响应或提示失败。

**解决方案**：

1. 检查计划名称是否符合要求
2. 确认日期范围设置是否合理（开始日期应早于结束日期）
3. 检查是否已达到计划数量上限
4. 刷新页面后重新尝试

#### 5.2.3 营养分析数据不准确
**问题描述**：系统显示的营养分析数据与预期不符。

**解决方案**：
1. 确认健康档案中的身高、体重等基础数据是否准确
2. 检查膳食计划中的食谱和份量是否准确
3. 考虑到个体差异，系统数据仅供参考，可根据实际情况进行调整
4. 如有异常，请联系系统人员反馈问题

### 5.3 系统性能问题

#### 5.3.1 页面加载缓慢
**问题描述**：系统页面加载时间过长或卡顿。

**解决方案**：
1. 检查网络连接是否稳定
2. 关闭浏览器中不必要的标签页和插件
3. 清除浏览器缓存和Cookie
4. 尝试使用其他浏览器
5. 在网络环境较好的情况下再次尝试

#### 5.3.2 操作响应延迟
**问题描述**：点击按钮或提交表单后，系统响应缓慢。

**解决方案**：
1. 确认网络连接状态
2. 避免在系统高峰期进行大量操作
3. 如长时间无响应，可刷新页面后重新操作
4. 检查系统公告，确认是否有系统维护或升级
